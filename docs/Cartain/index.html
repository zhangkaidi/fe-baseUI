# 1、图片形式弹层

  • 图片弹层形式，图片可替换；可以设置是否显示底部关闭按钮

# 2、普通展示弹层

  • 包括弹层内容和弹层标题，内容可动态配置；同时标题可配置是否显示；中间内容较多时支持滚动展示

# 3、底部确认/取消按钮弹层

  • 带有底部确认、取消按钮的弹层，其中标题、中部内容、底部按钮文案、按钮颜色可配置；

# 4、底部确认按钮弹层

  • 底部带有确认按钮的弹层，其中标题、中部内容、底部按钮文案、按钮颜色可配置；
